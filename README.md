# grunt-coffee-strip-code

A Grunt task to remove develop and test only code blocks from production builds from CoffeeScript files

## Getting started

#### Requirements

 - Grunt `~0.4.1` or above

    $ npm install grunt-coffee-strip-code --save-dev
